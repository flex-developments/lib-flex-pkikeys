# lib-flex-pkikeys
Librería que permite acceso a diferentes tipos de KeyStore, antes de acceder a las llaves permite establecer una lista blanca que filtre KeyStores dentro de la lista de confianza.

## Dependencias
 * lib-flex-helpers v010
   * commons-net-3.1
   * mail
